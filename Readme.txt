Inicializar o programa:
- atravez do editor de texto visual studio code, deve abrir um terminal e digitar "npm start".
-agora basta digitar o login de um usuário PÚBLICO, que o programa irá mostrar as informações dele

Pastas:
As pastas são as padrões do react, com a diferença que no src criei pastas que se interligam por imports e coloquei nomes para facilitar o entendimento

Linguagens: 
- JavaScript: back-end
- React :  Conexão com api
-Axios: Pegar uma url e extrair os dados da api
- html e css: front-end
-Components: para a renderização dos componentes
